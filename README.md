# AI Persona: Internal Communications Coordinator

## Overview
This persona transforms a generic AI assistant into a specialized Internal Communications (IC) partner. It is built for Comms managers, HR leaders, and People Ops teams who need to scale their messaging efforts without losing the "human touch."

## Key Features
- **Context-Aware Drafting:** The AI doesn't just write; it understands the difference between a Friday fun email and a Monday restructuring memo.
- **Tone Switching:** It automatically adjusts the voice depending on the channel (Email, Slack, All-Hands Script).
- **Strategic Alignment:** It checks every message against corporate values and business goals to ensure consistency.

## How to Use
### 1. Setup
Copy the `InternalCommsCoordinator.json` file into your AI's knowledge base or prompt library.

### 2. Prompting Examples
Here is how you should feed information to the persona:

- **Bad Prompt:** "Write an email about the new policy."
- **Good Prompt:** "Draft a Slack announcement regarding the updated remote work policy. The key changes are: (1) Mandatory Tues-Thurs in office, (2) Flex hours remain. The CEO is nervous about this, so keep it empathetic and emphasize collaboration benefits."

### 3. Iteration
If the output isn't perfect, use the feedback loop built into the persona:
> *"Change the tone to be more urgent and direct, as this is a safety update."*

## Why This Persona Works
- **Efficiency:** Cuts drafting time by 50%.
- **Consistency:** Ensures the CEO, HR, and Team Leads speak the same language.
- **Empathy:** Reminds the user to consider the employee experience, not just the information being shared.
